Battle Grid is tool to get some samples that are near target position. It can be userful if you want to test some world space positions that accessible from target position. For example if somebody see this position. Or if it near item or important swich. In agent there is collection of BattleGridPoint after it resive result of related function. BattleGridPoint have Position and Passability. It can be little outside NavMesh but always clipped to surface.
